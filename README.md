# 🌍 TravelNest

A responsive **TravelNest** landing page built using **HTML5** and **CSS3**. This project recreates the look and feel of a modern travel booking website while focusing on responsive design and clean UI development.

## 🚀 Live Demo

https://nsrinithi.github.io/travelNest/

## 📌 Features

* Responsive design for Desktop, Tablet, and Mobile
* Sticky navigation bar
* Hero banner with call-to-action
* Travel destination cards
* Experience and activity sections
* Beautiful image galleries
* Hover effects and smooth UI
* Modern footer design

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts
* Font Awesome Icons

## 💡 What I Learned

* Building responsive layouts using Flexbox
* Using media queries for mobile-first design
* Creating reusable card layouts
* Positioning elements with CSS
* Designing a modern travel website UI
